# 拼豆图纸工坊 (Perler Beads Blueprint Workshop)

拼豆图纸工坊是一个功能强大的在线工具，专为拼豆（Perler Beads）爱好者设计。它可以将您喜欢的任何图片转换为拼豆图纸，并提供了一套完整的像素编辑工具，让您能够轻松创作、修改和导出您的专属设计。

## ✨ 主要特性

*   **图片转图纸**：上传任意图片，自动转换为像素风格的拼豆图纸。
*   **智能图像处理**：
    *   **旋转与裁剪**：调整图片方向和构图。
    *   **对比度调整**：优化图片色彩，使转换后的图纸更清晰。
    *   **缩放**：自定义图片大小。
*   **强大的编辑工具**：
    *   🖌️ **画笔**：自由绘制像素点。
    *   🧹 **橡皮擦**：擦除不需要的像素。
    *   🪣 **油漆桶**：快速填充区域颜色。
    *   🧪 **吸管**：吸取画布上的颜色。
    *   🔄 **替换同色**：一键替换画布上所有相同的颜色，方便改色。
    *   ✋ **拖动**：在放大模式下轻松移动画布。
    *   🏁 **移除背景**：一键去除背景，保留主体图案。
*   **颜色管理**：
    *   🎨 **调色板**：提供丰富的拼豆常用颜色。
    *   支持自定义颜色选择。
*   **多种显示模式**：
    *   支持网格线显示/隐藏。
    *   ☀️/🌙 **明暗模式**：支持浅色和深色主题，保护视力，图标在不同模式下自动适配（黑/白/霓虹蓝）。
*   **导出与分享**：
    *   导出为高清图片。
    *   生成可打印的图纸。

## 🚀 如何使用

1.  **打开工具**：在浏览器中打开 `index.html`。
2.  **上传图片**：点击“选择图片”按钮，从本地上传一张您想制作的图片。
3.  **预处理**：
    *   使用左侧面板的工具对图片进行旋转、裁剪或调整对比度。
    *   设置目标尺寸（像素/豆子数量）。
4.  **生成图纸**：点击“生成”按钮，图片将被转换为像素图。
5.  **编辑完善**：
    *   切换到“图纸编辑”标签页。
    *   使用顶部工具栏的画笔、油漆桶等工具修改细节。
    *   如果需要，可以使用“移除背景”工具清理背景。
6.  **导出**：完成后，点击“导出”按钮保存您的图纸。

## 🛠️ 技术栈

*   **核心**：HTML5, CSS3, JavaScript (ES6+)
*   **样式**：Flexbox/Grid 布局，CSS Variables 实现主题切换。
*   **图形处理**：Canvas API 用于图像像素化和编辑操作。
*   **图标**：SVG (Scalable Vector Graphics)，支持 CSS 动态着色。

## 🎨 UI/UX 设计

*   **响应式设计**：适配桌面端和移动端操作。
*   **现代化界面**：简洁的工具栏，流畅的过渡动画。
*   **主题系统**：精心设计的黑夜模式，带有霓虹风格的高亮效果。
